AppCode

vmoptions

AppCode默认.vmoptions文件在~/Application/AppCode/bin/AppCode.vmoptions,更改该文件会改变AppCode的文件签名，应当使用Help-Edit Custom VM Options来更改配置：

# custom AppCode VM options
-Xss2m
-Xms256m
-Xmx4096m
-XX:NewSize=128m
-XX:MaxNewSize=256m
-XX:ReservedCodeCacheSize=192m
-XX:+UseCompressedOops

Xcode

Xcode 警告

  • 开启可信度较高的警告

Build Setting -> other c flags -> -Wall

  • 开启大部分警告

Build Setting -> other c flags -> -Wextra

  • 屏蔽第三方SDKwas built for newer iOS version警告

Target -> Build Setting -> Other Linker Flag -> -w
